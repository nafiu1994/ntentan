\subsection{Detailed Description}
A utility class for the \hyperlink{class_ntentan}{Ntentan} framework. This class contains the information necessary for routing. It also performs the routing of the pages and the loading of the controllers. This class also has several utility methods which help in the overall operation of the entire framework.
