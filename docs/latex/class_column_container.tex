\subsection{Detailed Description}
A special container which lays out its form elements in columns. The elements are evenly packed into the columns. The \hyperlink{class_column_container}{ColumnContainer} class is an extension of the \hyperlink{class_table_layout}{TableLayout} class which has one row but different columns based on the number of columns the container is supposed to have.
