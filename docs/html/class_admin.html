Detailed Description

Admin component provides an interface through which data in a model could be manipulated.

Adding this component to any controller automatically provides the interface for adding, editing and deleting items in the model class found within the attached controllers package or namespace. Apart from the manipulation interface, the admin component also provides a full blown admin console site.
